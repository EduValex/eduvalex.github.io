## Run locally

1. Install deps
   - Windows PowerShell:
     - `cd frameworks/astro`
     - `npm install`
2. Dev server
   - `npm run dev`
3. Build
   - `npm run build`
4. Preview
   - `npm run preview`

## Notes
- This is a minimal, fast static variant. You can progressively enhance with islands or integrate React components if needed using `@astrojs/react`.
